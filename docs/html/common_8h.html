Подробное описание

Публичный API протокола, таблиц и сетевых функций библиотеки распределённого интеграла.

Перечисления

◆ message_type_t

enum message_type_t

Идентификаторы сообщений бинарного протокола.

Тип сообщения хранится первым полем в каждой структуре протокола.

Предупреждения
Протокол отправляет нативные структуры C и значения double без преобразования порядка байт, поэтому в этом учебном проекте он рассчитан на однородные хосты.
Элементы перечислений
MSG_HELLO 

Клиент сообщает о себе и передаёт число ядер.

MSG_TASK 

Сервер отправляет одну задачу интегрирования и срез таблицы.

MSG_RESULT 

Клиент возвращает частичный интеграл для задачи.

MSG_STOP 

Сервер просит клиента завершиться или подтверждает успешное выполнение.

◆ enable_tcp_keepalive()

int enable_tcp_keepalive ( int  fd)

Включить TCP keepalive на подключённом сокете.

Аргументы
fdФайловый дескриптор TCP-сокета.
Возвращает
0 при успехе, -1 при ошибке setsockopt().

TCP keepalive является дополнительной страховкой. В проекте быстрые отказы отслеживаются прикладными таймаутами чтения, подключения и ожидания.
