Открытые атрибуты

uint32_t type
 Тип сообщения, обычно MSG_TASK или MSG_STOP.

 Идентификатор задачи, который worker возвращает в result_msg_t.
 
uint64_t begin_idx
 Первый глобальный индекс прямоугольника, входящий в задачу.

uint64_t points_count
 Число точек x/y, отправленных после этого заголовка.
 

Подробное описание

Заголовок задачи от сервера клиенту.

После заголовка передаются points_count значений double для x[], а затем points_count значений double для y[].
